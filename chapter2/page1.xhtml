2.1 - Your First HTML Document

HTML is the language of the Web. Here we will show how to create a basic HTML document.

  1. Open up your favorite text editor or code editor and create a new file with the extension .html.
  2. Give it the following content:
    <!DOCTYPE html>
    <html lang="en">
    <head>
    <title>A title</title>
    </head>
    <body>
    </body>
    </html>
  3. This is the required content for an HTML document. To learn about each thing, see the next pages.

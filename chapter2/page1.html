The DOCTYPE

Every web page must have a DOCTYPE. It is <!DOCTYPE html>.

The html tag

Every web page must also have the html tag. This tag wraps around everything except for the DOCTYPE. It has a required lang attribute where you must state the language of the webpage, like en

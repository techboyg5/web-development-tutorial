The Code Editor dialog (which you're looking at right now) has four tabs:

  • HTML
  • CSS
  • JavaScript
  • Preview

To use the Code Editor, enter HTML, CSS, and JavaScript code in their respective tabs. To see what the code would look like, move over to the Preview tab and click the "Update Preview" button.

If you'd like to embed the Code Editor on your own site, you can use the following code:

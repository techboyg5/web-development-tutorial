2.2 - DOCTYPE

You need <!DOCTYPE html> to be the first thing in every HTML document. This is called the document type declaration, or the DOCTYPE.

Why?

This triggers standards mode in web browsers. Without the DOCTYPE, web browsers will display the page in quirks mode.

These DOCTYPEs are outdated. Do not use them.

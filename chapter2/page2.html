2.2 - DOCTYPE and html tag

You need <!DOCTYPE html> to be the first thing in every HTML document. This is called the document type declaration, or the DOCTYPE.

Why?

This triggers standards mode in web browsers. Without the DOCTYPE, web browsers would display the page in quirks mode.

Note: You might see some other DOCTYPEs occasionally. Avoid using these DOCTYPEs as they are outdated.

The html tag

With the exception of the DOCTYPE, all content in your page must be wrapped around the html tag.

The html tag must have a lang attribute on it to declare the main language of the page. For English, use lang="en".

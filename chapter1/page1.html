Welcome to the brand-new Web Development Tutorial! Here you'll learn how to create a beautiful website.

What is this?

This is a Web Development Tutorial by techboyg5, a project and website lover.

Why was this created?

I know that there are many people out in the world who don't know how to code. Or need help.

The most popular resource for web development became MDN Web Docs, or MDN.

The problem was, MDN basically became the resource for learning web development. But I know that people love freedom and choice.

So I came up with a solution to the problem. I made a Web Development Tutorial of my own, which will make learning how to make websites fun and easy.

What will you learn?

In this tutorial, you'll learn the following things:

  • HTML
  • CSS
  • JavaScript

Requirements

Interested in what you need to start the tutorial? Read below.

  • A PC
  • An internet connection
  • Some time to breathe

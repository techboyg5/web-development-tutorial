1.1 - Using This Tutorial

Welcome to the Web Development Tutorial! You are here now, but you might not know how to use this! As a result, here is how you use the tutorial.

Views

There are three different views, each described below.

Frames View

In frames view, there are three different frames. On the top left are the chapters. On the bottom left are the pages in that chapter. On the right is the page.

Detachable Frames View

This view is the same as frames view, except that each of the frames is a window.

No Frames View

In no frames view, the page occupies the entire browser window.

Footer

The footer on each of these pages is always the same. The following links are on the footer:

  • Previous/Next Page
  • Frames/No Frames
  • Table of Contents (intended for no frames view, but also works in frames view)
  • Join my Discord server
  • Contact me
  • Link to GitHub repository

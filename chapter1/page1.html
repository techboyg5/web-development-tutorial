Welcome to the brand-new Web Development Tutorial! Here you'll learn how to create a beautiful website.

Requirements

To develop Web sites:

You need an Internet connection both to read this tutorial and to actually do the web development.

You need Microsoft's Visual Studio Code installed on your computer. You can download Visual Studio Code here.

Finally, you need a GitHub account. You can create one here.

Once you verify you're connected to the Internet, have downloaded VS Code and have a GitHub account, you can continue to the next page.
